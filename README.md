# EMG Controlled Prosthetic Hand

This project demonstrates a prosthetic hand controlled using EMG muscle signals.

## Components
- Arduino Uno
- MyoWare EMG sensor
- Servo motors
- Li-ion battery

## Working Principle
The EMG sensor detects muscle activity and sends analog signals to the Arduino.  
When the signal crosses a threshold, the Arduino moves the servo motors to simulate finger movement.
